Stock

When user tapped on Stock icon on the home page, will be able to see the available stock information of the store. User can see all the Stock information. User may search the stock information by Product name, SKU ID, Quantity available in store and Price of the product

A) Critical Stock :
The Stock which is about to finish in the store is treated as Critical stock. The available quantity is below the Re-Order point.
The Store Manager or the person at the counter can verify the stocks. If the stock goes down below the critical level, the system sends notification to specific roles. The user can see both critical stocks and normal stocks. If the user wants to search for specific stock he/she can enter SKUCode or Product name and get the update on the stock.

B) Normal Stock :
All The other Stock, for which is The available quantity is above the Re-Order point is treated as Normal Stock.
The Store Manager or the person at the counter can verify the stocks. If the stock goes down below the critical level, the system sends notification to specific roles. The user can see both critical stocks and normal stocks. If the user wants to search for specific stock he/she can enter SKUCode or Product name and get the update on the stock.
