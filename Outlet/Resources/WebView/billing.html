Billing

A) New bills :

1) User can enter SKU ID manually in the text box or scan the barcode using camera, by tapping scan icon
2) Once a barcode scanned or entered, will be added the product to list with the the details (SKU ID, Product name, Quantity, Price)
3) Adding multiple products will display in different rows
4) Adding Multiple products having same ID will multiply the quantity and display the total price
5) At anytime while adding the products, user can delete any specific product from the list
6) Proceed to Payment:- This option is displayed only after adding a product to the list else it will be hidden. This option goes invisible once again on deleting the products list manually. Selecting this options takes to the ‘Payment options’ screen where it displays the total amount to be paid and the available options to finish the sale

a) By Cash
b) By Credit card / Debit card


Cash :

This is the default payment option to finish the sale. User can directly select “Generate Bill” once the cash the received. Created sale may be sent to customer in the form of SMS or Email which are optional.

Credit card / Debit card:

This is the second type of payment method, wherein user
(i) can swipe his card and makes the payment or
(ii) can enter card details manually and makes the payment.

B) Past bills

1) Feature to enter the sale ID, which is provided for the successful bill transaction.
2) By tapping the open bill icon, User can see the billing details.
3) User may edit the quantity there itself and re-generate the bill again.
4) If user wants to make a fresh bill along with the items present in the old bill (Used to add more items to the billing list), can go back to the home page and select the new bill icon.
5) Now user can do all the manipulations as there in the new bill screen, for a past bill.
